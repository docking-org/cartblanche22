Search by ZINC identifier, one per line

You can get result with different formats. If you want to specify output fields, you can add output_fields. The example below returns only smiles, zinc_id by default. If output_fields are not specified , all possible fields are returned. Please note that there are always smiles and zinc_id fields by default.

/*

CURL commands for searching single ZincID

Example: curl https://cartblanche22.docking.org/search.txt -F zinc_id=ZINCms000002NiP3 -F output_fields='smiles,zinc_id'

Description Attributes Possible fields
To specify return format curl https://cartblanche22.docking.org/search.txt
  • .txt
  • .csv
  • .json
To add search value -F zinc_id=ZINCms000002NiP3 Any single zinc identifier
To specify output field
If it doesn't defined, search will return all possible fields 		-F output_fields='smiles,zinc_id'
  • catalogs
  • smiles
  • sub_id
  • supplier_code
  • tranche
  • tranche_details
  • zinc_id
*/

CURL commands for searching multiple ZincIDs

Example: curl https://cartblanche22.docking.org/substances.txt -F zinc_id-in=@test.txt -F output_fields='smiles,zinc_id'

Description Attributes Possible fields
