CURL commands for searching ZincIDs

Below curl command returns a json file with all possible fields.

If you want to curl with text file:

curl https://cartblanche22.docking.org/substances.json -F zinc_id-in=@/nfs/exd/zinc-22/H16/H16P220/H16P220-N.txt

Curl without text file (ZincIds separated by ,):

curl https://cartblanche22.docking.org/sublist -F zinc_id-in="ZINCgs0000000OAi,ZINCgs0000000OAj"

If you want to learn more about search, please go to Zinc22 documentation on wiki page
