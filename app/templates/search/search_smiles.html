CURL commands for searching Smiles

Below curl command returns a json file with all possible fields.

If you want curl with text file:

curl https://cartblanche22.docking.org/smiles.json -F smiles-in=@smiles.txt -F dist=0

Curl without text file (smiles separated by ,) :

curl https://cartblanche22.docking.org/smilelist -F smiles-in="CC(=O)OCOC(=O)c1cscc1Nc1cccc(C)c1Cl,C1=CC=CC=C1C2=CC=CC=C2" -F dist=0

If you want to learn more about search, please go to Zinc22 documentation on wiki page
